# IQ2

## Installation ⚙️

### Requirements:
 - Python 3.8.10

### Virtual Env Setup 👨‍🏭
```
python -m venv venv
```
For Windows:
```
./venv/Scripts/activate
```

For Linux:
```
source activate venv/bin/activate
```

### Install libs 👩‍🏭
```
cd DSTA-Assignment1
pip install -r requirements.txt
```
